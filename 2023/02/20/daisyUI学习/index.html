daisyUI的安装

环境：需要安装Node.js和Tailwind CSS。
node环境安装不再多说，先安装tailwindCSS，这里介绍脚手架安装

安装Tailwind CSS

  1. 安装Tailwind CSS 通过npm安装 并创建tailwind.confing.js文件
  2. 在tailwindcss.config.js配置模板路径（不要改）
  3. 将三大部件写到主css文件中
  4. 启动
  5. 在html中使用 到这里安装已经结束，按照官网一步步走，不过有几个注意的点，在第四步中，
    官网中是 “tailwindcss -i ./src/input.css -o ./dist/output.css –watch”。
    我在脚本中配置的是 “tailwindcss -i ./src/input.css -o ./src/dist/output.css –watch”
    一定要注意配合node+express使用时指定的静态资源路径。
    在修改之前一直报MIME type (‘text/html‘) is not a supported stylesheet MIME type, and strict MIME checking is enable错误，不要以为真的是type错误，大多数情况下都是静态资源路径错误。 从服务器中指定的静态资源文件例如app.use(express.static('src'))出发的

tailwindcss使用

他的功能类十分齐全，我们可以不用编写css，但带来的问题是叠一大堆的类名，但他的优点在于：

  • 没必要为类名浪费精力
  • css停止增长
  • 更改会更方便安全
